2. 🤖 **AI不是万能的** - 预测仅供参考，不保证准确

## ✨ 功能特性

### 🧠 AI预测引擎
- **6种神经网络模型**: LSTM、Transformer、GRU、TCN、Mamba、混合模型
- **80+技术指标**: 全面的技术分析
- **置信度评估**: 知道何时不应交易
- **自动学习**: 从互联网自动搜索并学习

### 🔍 自动搜索
- 涨跌幅排行榜
- 热门股票
- 机构推荐
- 行业龙头
- 放量突破

### 💹 交易系统
- **模拟交易**: 无风险练习
- **真实交易**: 支持同花顺等券商
- **风险管理**: 自动止损、仓位控制
- **T+1合规**: 符合A股交易规则

### 🖥️ 专业界面
- 实时行情图表
- 信号可视化
- 投资组合追踪
- 交易历史记录

## 🚀 快速开始

### 30秒快速体验

```bash
# 1. 克隆项目
git clone https://github.com/yourusername/ai-stock-trading.git
cd ai-stock-trading

# 2. 安装依赖
pip install -r requirements.txt

# 3. 验证安装
python main.py --check

# 4. 一键自动学习（AI自动搜索互联网并训练）
python main.py --auto-learn

# 5. 启动应用
python main.py
